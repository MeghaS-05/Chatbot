Description
An AI-powered chatbot that reads PDF documents and answers user questions based on their content.

Features
1) Upload and process PDF documents
2) Ask questions in natural language
3) Context-aware answers from document content
4) Fast and efficient text retrieval

Tech Stack
Python
NLP / LLM techniques
PDF processing libraries (like PyPDF)

Usage
Run the application: python app.py
Upload a PDF file
Ask questions in the chat interface
Get answers based on the PDF content
